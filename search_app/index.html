kogitas (demo)

Welcome to kogitas : the Search engine for real knowledge

This demo is an approximation to what will happen when someone queries "Renaissance." As you can see, Kogitas generated a graph of people, places, things, and events related to the renaissance. In effect, the query is to explore the renaissance. Kogitas does not presuppose a correct answer to the query. Currently, I am working on a more sophisticated set of tools that will allow Kogitas to become a general purpose search engine.
