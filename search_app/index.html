kogitas (demo)

Welcome to kogitas : a new way to explore knowledge

This demo has some pre-selected queries. Soon, you will be able to make whatever query you want - not just a couple I have manually created for demo purposes! As you can see, Kogitas generated a graph of people, places, things, and events related to the query. In effect, the query is to explore the renaissance. Kogitas does not presuppose a correct answer to the query. Currently, I am working on a more sophisticated set of tools that will allow Kogitas to become a general purpose search engine.
